 [![Build Status](https://api.travis-ci.org/nodayoshikazu/jsonselect-cli.png)](http://travis-ci.org/nodayoshikazu/jsonselect-cli)


#Installation
 
    > npm install
    > make build

 To run the tests:

    > make test

    > npm test
